STM32-PWM-DC-Motor-Control



用 STM32F103C8T6（蓝色小板子）配合按键和 OLED，控制直流电机 速度 和 正反转，OLED 实时显示当前速度值（带 + / - 号，负就是反转）。



功能



· 按键1：加速（每次 +20，范围 0 \~ +100）

· 按键2：减速/反转向（速度为正时减速，减速到 0 后再按就变成反向加速，范围 0 \~ -100）

· OLED 显示速度，比如 +80、-40

· 软件消抖 + 松手检测，按一次只响应一次，不会乱跳

· 正反转切换平滑，没有瞬间硬冲击



硬件接线



STM32 引脚 接什么

PA6 (TIM3\_CH1) TB6612 驱动模块的 PWMA

PA4 TB6612 的 IN1

PA5 TB6612 的 IN2

PB1 按键1（加速）

PB11 按键2（减速/反向）

PB6/PB7 OLED (I2C)

GND 所有 GND 连在一起



电源注意：



· STM32 通过 USB 供电

· 电机必须单独供电（比如 5V 电池或 5V/2A 适配器），不要共用 STM32 的 USB，否则下载程序会失败，电机转速也上不去。



代码结构





├── Motor.c/h         // 电机方向 + 速度函数

├── PWM.c/h           // 定时器 PWM 初始化，设置占空比

├── Key.c/h           // 按键扫描（消抖、松手检测）

├── OLED.c/h          // OLED 显示

└── main.c            // 主循环：读按键，调速度，刷新显示



核心函数就一个：





// speed: -100 \~ +100，负数是反转

void Motor\_Setspeed(int8\_t speed);



怎么跑起来



1\. 把代码下载到 STM32（用 ST-Link）

2\. 按硬件接线表连好电机驱动和电机

3\. 电机独立上电

4\. 按按键1加速，按键2减速/反向，看 OLED 上的数字变化



已知问题



· 如果电机电源电流不够（比如只有 500mA），速度加到 80 左右就上不去了，而且几秒后会掉速，STM32 的红灯会变暗。这是供电不足，换个大电流电源就好。

· 下载程序时如果电机也接着电，偶尔会连接失败。可以下载时先把电机电源断开，下完再插上。
